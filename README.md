# Project Overview

This project is a biography application that provides detailed information about Suraj Singh, showcasing his life, work, and achievements.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rammyyadav-dot/Suraj-Singh-Bio.git
   ```
2. **Navigate into the directory**:
   ```bash
   cd Suraj-Singh-Bio
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the application**:
   ```bash
   npm start
   ```

Enjoy exploring the life and achievements of Suraj Singh!
